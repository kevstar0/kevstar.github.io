Installation procedure

sign up/ sign into Mobile Fly
request connectivity
NOTE: without a mobile fly account your connecitivity request will be ignored.
Anonymous users have limited access

download and install git into your computer
create a new folder into your computer
use your terminal and cd into the new folder
clone this repository using git
install npm with the command "sudo apt-get npm" or download npm from the official site
your computer will install Node into your computer
